Install R and RStudio:

First, you will need to install R and RStudio (other IDEs will not be supported). For those who are unfamiliar, R is the name of the programming language we will be using and RStudio is a convenient interface / development environment for the R language.

  • Install R : Install RStudio: Go to https://www.rstudio.com/products/rstudio/download/ and click on the link for your operating system.

  • Install RStudio : Go to https://www.rstudio.com/products/rstudio/download/preview/ and click on the installer link for your operating system. Note that you need this updated preview version for labs in this course, so please make sure to install this version even if you have previously installed RStudio.

See this video for step-by-step installation instructions if needed.

NOTE : Even if you already have R and RStudio installed, you should still visit these links to confirm that you have the most recent versions of these software installed.

Install R Packages:

1. Install packages:

Install the packages we will use in this lab.

Type the following commands in the Console: 

install.packages("tidyverse")
install.packages("statsr")
install.packages("PairedData")
install.packages("dplyr")
install.packages("ggplot2")

2. Load other packages: 

library(statsr)
library(PairedData)
library(dplyr)
library(ggplot2)

The use of these packages will be discussed in detail in the lab.
